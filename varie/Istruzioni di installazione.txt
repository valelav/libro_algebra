In caso di aggiornamento di Python
1. Disinstallare completamente versione precedente
2. Rimuovere completamente eventuali residui in Appdata/Roaming e in Appdata/local/Programs
3. cancellare evenutali indicazioni al path nelle variabili d'ambiente
4. Reinstallare Python (selezionando la casella per l'aggiornamento del path nella variabile d'ambiente)
5. Installare i moduli necessari pip install lxml 
			         pip install requests 
				 pip install pdfCropMargins
